# BenLey-UNIXProject

The purpose of this project is to create a personal media server using html. The fileloop script loops through the folders in a given directory and creates a new html file for each movie. It also adds a link to this new file with an image of the movie poster, and adds this to the "PastetoIndex.txt" file. After looping through every folder, the index file is started, the paste file is concatenated in, and then the index file is ended.
